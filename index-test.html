Welcome to Code Crunch

Code Crunch bridges global innovation with local engagement—offering a dynamic platform for all university students, no prior experience required! Join us to break down barriers in technology, participate in hands-on coding workshops, collaborate on community projects, and prepare for the future with immersive virtual and on-campus experiences.

Whether you join via virtual sessions or in person at FIU, we provide the resources, mentorship, and community to help you grow as a coder and innovator.

Let’s innovate together! Built and led by students, for students.

Weekly Access Updates

Each Code Crunch course has its own GitHub repository. Weekly files, including Google Slides, code blocks, and other resources, are uploaded to ensure members stay up to date.

Access to these resources is available exclusively to members.

Note: Access to all Google Docs and Slides will be granted using the Google account provided in the "Become a Member" form.

GitHub Repositories

After becoming a member, you will receive an invite to our GitHub repositories.

  • Each course has its own repository.
  • You should see a repository for each course when accessing the Code Crunch organization on GitHub.
  • Weekly updates include slides, code blocks, and session resources.
